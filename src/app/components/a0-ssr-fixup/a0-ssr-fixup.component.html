Wrap and mock Auth0 Service

Auth0 is meant to authenticate users which is not required when pre-rendering on the server.

In previous versions of Auth0 it was sufficient to guard against calling Auth0 methods on the server using isPlatformBrowser or isPlatformServer, but as from v2.0.1, this is insufficient.

Ensure AuthModule is not imported on the server

  • Create an AuthClientService to wrap the Auth0 Service (not provided in 'root')
  • Create an AuthServerService to mock the AuthClientService for the SSR prerendering (not provided in 'root')
  • Remove AuthModule from AppModule
  • Make an AppClientModule, which imports both AppModule and AuthModule. Provides array includes AuthClientService
  • Make an AppServerModule which provides the AuthServerService in place of AuthClientService
